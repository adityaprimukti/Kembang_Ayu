# Kembang ayu
The Kembang Ayu project is a web application developed to provide online information and booking services for beauty spas and body treatments. Through this application, users can view various types of treatment services, obtain detailed service information, and make reservations more easily and efficiently. This project was developed as part of the Certified Independent Study and Internship (MSIB) Batch 6 program at PT Mitra Talenta Group, with a focus on developing responsive and user-friendly web applications.

## ✨ Features
- Displays a catalog of spa and beauty treatment services
- Detailed treatment service pages
- Responsive and user-friendly website design
- Service data management system using a database

## 🛠 Tech Stack
- **React.js** – Frontend framework for building single-page applications
- **Tailwind CSS & CSS** – Styling and creating responsive displays
- **MySQL** – Database for storing service data and related information

## 🚀 Cara Menjalankan Project

### 1. Clone Repository
```bash
git clone https://github.com/Tirtayudha-W/project-skripsi.git
```

### 2. Running Frontend
```bash
npm install
npm start
```

### 3. Running Backend
```bash
cd server
node server.js
```
